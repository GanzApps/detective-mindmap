# UI Reference: MAC Video Alignment

## Source

- `.planning/references/mac-ui-reference/2026-04-09.mp4`

## Shell Direction

- Calm, productized shell rather than a raw dense tool
- Centered framing with generous whitespace
- Restrained surfaces and borders
- Purple used for primary actions and active states
- Light and dark modes should both preserve hierarchy and calmness

## Layout Bands

- Top shell / page header
- Left utility rail
- Center hero graph workspace
- Right analysis surface
- Bottom timeline
- Bottom command surface

## Behavioral Interpretation

- The graph remains the hero surface, but it is framed by structured rails and bands
- Detail/result surfaces should not heavily cover the graph
- Tabs and filters read as product navigation, not debug controls

## Theme Guidance

- Light mode is the calm baseline
- Dark mode is restrained, not neon
- Shell tokens should stay separate from graph/entity semantic colors

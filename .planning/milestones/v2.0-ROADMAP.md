# Milestone Archive: v2.0 Investigation Workspace Revamp

**Version:** v2.0  
**Completed:** 2026-04-14  
**Commits:** 70  
**Phases:** 5 (8–12), 18 plans total  

## Accomplishments

1. **Unified shell design tokens** — Dashboard and workspace now share a single visual language with light/dark theme support and purple accent semantics.
2. **Investigation workspace framing** — Tabbed left rail (Raw Evidence / Filters & Layers), right-side on-canvas analysis panel, persistent minimap, and integrated timeline dock.
3. **Reference-aligned graph visual language** — Entity-type-specific iconic node shapes, semantic color coding, typeahead-first search, and dimmed focused-network readability.
4. **2D / 3D interaction parity** — Shared selection model, preserved placement intent across view switches, gesture parity between renderers, and consistent drag behavior.
5. **AI command routing foundation** — Known-intent command parser, mixed static/context-aware quick chips, inline execution feedback, and AI-result routing into the right-side panel.

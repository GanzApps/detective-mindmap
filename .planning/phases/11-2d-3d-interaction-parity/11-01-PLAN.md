# Phase 11-01 Plan: Shared Placement State

## Goal

Introduce one shared node placement state that both renderers consume and update.

## Tasks

1. Add a shared node position map in `GraphWorkspace`.
2. Feed stored positions into `ForceGraph2D` and `MindMap3D`.
3. Emit placement updates from renderer drag interactions back to `GraphWorkspace`.
4. Keep selection and highlighted state untouched while switching renderers.

## Files

- `src/components/graph/GraphWorkspace.tsx`
- `src/components/graph/ForceGraph2D.tsx`
- `src/components/graph/MindMap3D.tsx`

## Done When

- exact node placement carries across view switches
- view switching does not reset selected/focused state

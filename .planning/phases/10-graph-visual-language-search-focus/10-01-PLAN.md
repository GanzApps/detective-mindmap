# Plan 10-01: Search Commit Flow

## Goal

Separate query typing from committed graph focus so search behaves as a true typeahead-first experience.

## Tasks

1. Refactor graph search state in `GraphWorkspace.tsx` into:
   - current query text
   - suggestion results
   - committed selected search target
2. Update `ForceGraph2D.tsx` to render suggestion UI and invoke committed selection callbacks instead of auto-focusing from every keystroke.
3. Ensure committed search selection drives:
   - focus emphasis
   - selected node state
   - right-panel-compatible active context
4. Add tests covering:
   - typing does not auto-focus
   - selecting a suggestion applies focus

## Exit Criteria

- Search behaves as dropdown/typeahead first
- Focus is only applied after user selection
- Tests cover the split between query state and committed focus state

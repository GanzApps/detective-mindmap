# Plan 10-03: Focus Readability and Validation

## Goal

Tune default readability so the graph remains information-rich without becoming noisy, then validate the whole app.

## Tasks

1. Refine focused-network dimming behavior for nodes and edges so unrelated elements stay visible but visually secondary.
2. Curate default label density:
   - node labels on by default
   - edge labels selective, especially for focused paths or sufficient zoom
3. Verify the right-side analysis panel still reads clearly with the new graph semantics.
4. Run full validation and close the phase with execution-ready evidence.

## Exit Criteria

- Default graph remains readable with labels on
- Edge labels are selective and useful
- Full test suite and production build pass

# Plan 10-02: Semantic Node Styling

## Goal

Introduce the approved iconic shape and semantic color system, prioritizing strong 2D readability and pragmatic 3D carryover.

## Tasks

1. Extend shared graph styling helpers in `forceSimulation.ts` and related utilities for semantic:
   - colors
   - shape metadata
   - label/chip emphasis rules
2. Rework 2D node drawing to use iconic outer silhouettes rather than same-shape nodes with letter glyphs.
3. Carry semantic color and dimming treatment into the minimap output and pragmatic 3D rendering updates where technically straightforward.
4. Add tests or snapshot-level assertions for semantic renderer output seams where practical.

## Exit Criteria

- 2D graph uses distinct iconic silhouettes per entity family
- Semantic colors follow the approved mapping
- Minimap and focused states inherit the same semantic styling model
